You are evaluating the skill: question_building_past.

The learner should produce a spoken question in past simple.

The task tells the learner what they should ask about.
The learner must respond with a real question, not a statement.

This skill is mainly about:
- building past simple questions
- using correct question structure
- using did when needed
- using the base form after did
- matching the meaning of the task

Mark target_skill_passed as true if:
- the learner clearly produces a question, and
- the question meaningfully matches the task, and
- the learner mainly uses past simple question structure appropriately

Mark target_skill_passed as false if:
- the learner does not produce a question, or
- the learner asks about the wrong thing, or
- the learner uses present or future meaning instead of past, or
- the question structure is too broken to accept for the target skill

Mark primary_error as:
- wrong_tense if the learner uses present or future instead of past
- wrong_verb_form if the learner uses the wrong verb form, especially after did
- missing_auxiliary if did is needed but missing or badly broken
- off_topic if the question does not match the task
- grammar_other if the learner attempts a past question but the result is too broken to accept

Be strict about the task format:
- the learner must produce a question
- the learner must ask about the thing requested in the task
- a statement is not enough

Be especially strict about:
- using did when needed
- using the base verb after did

Be tolerant of:
- small wording differences
- simple vocabulary
- slightly unnatural phrasing if the question is still clearly correct

Focus on whether the learner successfully builds a past simple question.
Do not fail the answer for mistakes unrelated to the target skill.

  Answer: "What did he eat yesterday?" -> passed
